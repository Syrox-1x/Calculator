# Simple Calculator

A lightweight, user-friendly calculator designed to perform basic arithmetic operations quickly and accurately. 

## 🚀 Features

*   **Addition (`+`)**: Adds two or more numbers.
*   **Subtraction (`-`)**: Subtracts one number from another.
*   **Multiplication (`*`)**: Multiplies numbers together.
*   **Division (`/`)**: Divides numbers with safety checks to prevent division by zero.
*   **Clear Function**: Instantly resets the current calculation.
